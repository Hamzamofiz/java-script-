What is JavaScript?

JavaScript is a programming language that runs in web browsers and on servers (with Node.js). It lets web pages become interactive: responding to clicks, changing content dynamically, validating forms, and much more. Unlike HTML (structure) and CSS (style), JavaScript is for behavior and logic.

Key points

  • Runs in the browser: interacts with the page via the DOM (Document Object Model).
  • Dynamic & flexible: change text, styles, respond to events, fetch data from servers.
  • Multi-paradigm: supports functional, procedural, and object-oriented styles.
